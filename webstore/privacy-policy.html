findEmm Privacy Policy

Effective date: 2026-07-16

findEmm is a local-first Chrome extension for recruiter-directed prospect research.

Data we handle

findEmm handles prospect details that a recruiter enters or explicitly captures from the active page, including names, company details, public profile URLs, work email addresses, business phone numbers, notes, lists, and source evidence. It also handles the local API pairing token entered by the recruiter.

Use and sharing

Data is used only to provide prospect research, record management, export, and draft-outreach features. findEmm does not operate a hosted collection service, sell data, use data for advertising, train models on it, or allow human review of user data. Lookup data is sent only when the recruiter requests research, to the recruiter-run local API at 127.0.0.1 and to recruiter-authorized sources.

Storage and security

Saved records and the local API pairing token are encrypted in Chrome local storage with AES-GCM. The key is derived from the user’s vault passphrase with PBKDF2 and is held only while the popup is open. The passphrase is never stored or transmitted.

Retention and deletion

Records remain on the device until removed or Chrome extension storage is cleared. Local API batches disappear when the local API stops.
